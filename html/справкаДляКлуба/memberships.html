Абонементы

На странице с абонементами Вы можете просмотреть все доступные абонементы

 

 

Фильрация абонементов

Для выбора критерия для поиска из выпадающего окна соответствующего критерия. Выберите подходящий пункт списка и кликните на него. После Вы увидите только те абонементы, которые соответствуют Вашему выбору.

 

 

 

Выбор абонементов по цене

Для того, чтобы посмотреть абонементы с ценой от 0 грн. до желаемой стоимости необходимо переместить бигунок до необходимой отметки. Для этого нажать на булый элипс и удерживая нажатую левую кнопку мыши передвигать влево или вправо.

 

 

 

Оформление абонемента

Для того, чтобы оформить абонемент кликните на кнопку оформить под абонементом, котлрый Вы выбрали

 

 

После нажатия откроется форма для заполнения данных.

Правила заполнения :

•Имя и фамилия должны содержать только буквы.

•Электронная почта gmail

•Номер теелфона в формате 0999999999

После введения всех данных нажмите на кнопку "оформить"

 

 

Если данные введены правильно вы увидите окно с сообщением об успешном оформление и предложение зарегистрироваться. Если Вы готите зарегистрироваться кликните на кнопку регистрация. Если нет - кликните на кнопку позже. Инструкции для регистрации читайте здесь.
